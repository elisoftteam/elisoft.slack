# Elisoft.Slack

A lightweight and simple .NET library for sending notifications to Slack using Incoming Webhooks in requests.

✔ .NET 9  
✔ Dependency Injection friendly  
✔ ILogger support  
✔ Fully testable  
✔ Minimal dependencies  

---

## Installation

Install via NuGet:

```bash
dotnet add package Elisoft.Slack
